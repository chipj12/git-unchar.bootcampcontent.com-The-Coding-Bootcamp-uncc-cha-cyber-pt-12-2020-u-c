## Bash Commands 
| Command | Definition|
| :------ | :-------: |
| ls | Show directory contents, lists names of files |
| cd | Change directory |
| mv | move |
| cp | copy |
| su | switch user |


## Directory Structure
| Directory | Description |
| :-------- | :---------: |
|  /| root |
| ~  | home |
| .  | current directory |
| .. | previous directory |
